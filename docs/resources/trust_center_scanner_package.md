# snowflake_trust_center_scanner_package (Resource)

Resource for managing Trust Center scanner package configuration. For more information, see [Trust Center](https://docs.snowflake.com/en/user-guide/trust-center).

## Example Usage

```terraform
resource "snowflake_trust_center_scanner_package" "security_essentials" {
  scanner_package_id = "SECURITY_ESSENTIALS"
  enabled            = true
  schedule           = "USING CRON 0 2 * * * UTC"

  notification {
    notify_admins      = true
    severity_threshold = "High"
    users              = ["SECURITY_ADMIN"]
  }
}
```

-> **Note** If a field has a default value, it is shown next to the type in the schema.
