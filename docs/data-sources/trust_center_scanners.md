# snowflake_trust_center_scanners (Data Source)

Data source for listing Trust Center scanners. For more information, see [Trust Center](https://docs.snowflake.com/en/user-guide/trust-center).

## Example Usage

```terraform
data "snowflake_trust_center_scanners" "all" {}

data "snowflake_trust_center_scanners" "security_essentials" {
  scanner_package_id = "SECURITY_ESSENTIALS"
}
```
